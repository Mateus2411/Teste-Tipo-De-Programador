# Dependencias

- Node.js v14 o superior ou yarn v1.22 o superior
- npm v6 o superior

## Inicializar o projeto

- Nessa mesma pasta do projeto, rode o comando:

```bash
npm install
```

- Agora para começar a rodar, rode o comando:

```bash
node server.js
```

- No terminal verá a mensagem:

```bash
Servidor rodando na porta 5000
```

- Pronto agora o backend está rodando!

***Se ja tiver iniciado o frontend, IGNORE***
***Caso nâo iniciou o frontend, abra outra aba no VSCode com a pasta frontend vue***

- **Siga para o README.md do frontend vue**
